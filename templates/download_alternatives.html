🎬 Alternativas de Download

YouTube bloqueia downloads diretos de servidores. Aqui estão suas opções:

💻 Instalar Localmente (Recomendado)

A forma mais confiável: execute o app em sua máquina para downloads ilimitados.

git clone https://github.com/victorhuggomed2006-ux/YoutubeDownloader cd YoutubeDownloader python -m venv venv venv\Scripts\activate pip install -r requirements.txt python app.py

Abra http://localhost:5000 no navegador após executar os comandos.

⚙️ Usar Terminal (yt-dlp)

Se já tem yt-dlp instalado, use este comando diret no terminal:

yt-dlp -f "{{ quality }}" "{{ url }}"

Instale yt-dlp com: pip install yt-dlp

🌐 Serviços Online Alternativos

Use sites que permitem download de YouTube:

🔗 Y2Mate 🔗 SaveFrom.net 🔗 Loader.to 🔗 YouTube-Downloader

⚠️ Estes serviços podem ter anúncios. Use com cuidado.

ℹ️ Por que isso acontece?

YouTube bloqueia datacenter IPs (Vercel, Heroku, etc) com desafios de bot. Isso é uma medida de segurança do YouTube, não uma limitação do nosso app.
